## 1. 实验目的
- 模拟一元线性回归数据：y = 1 + 2x + ε，其中 $\epsilon \sim N(0,1)$。
- 使用最小二乘法公式手动估计回归系数 β̂₀  和 β̂₁。
- 计算估计量的偏差和方差。
- 对比手动计算与 sklearn、statsmodels 的结果。
- 进行假设检验（β₁ = 0）和整体模型的 F 检验。
- 绘制 β̂₁ 的分布图，验证大样本性质。

## 2. 数据生成过程
- 真实参数：β₀ = 1，β₁ = 2。
- 样本量：n = 100。
- 自变量 X 在 $[0, 10]$ 上均匀分布。
- 误差项 $\epsilon \sim N(0, 1)$。
- 随机种子设为 42，保证结果可复现。

## 3. 结果

### 3.1 手动估计量的偏差与方差（基于 1000 次模拟）
| 指标 | 数值 |
|------|------|
| 真实 β₁ | 2.0000 |
| 估计期望 | 2.0005 |
| 偏差 | 0.0005 |
| 方差 | 0.001241 |

### 3.2 三种方法估计结果对比（单次模拟示例）
| 方法 | β₀ | β₁ | R² (仅 sklearn) |
|------|----|----|-----------------|
| 手动公式 | 0.9487 | 2.0077 | - |
| sklearn | 0.9487 | 2.0077 | 0.9689 |
| statsmodels | 0.9487 | 2.0077 | - |

### 3.3 假设检验与方差分析（基于 1000 次模拟的平均结果）
| 检验项 | 统计量 (平均) | p-value (平均) |
|--------|---------------|----------------|
| β₁ = 0 的 t 检验 | - | 5.2812e-69 |
| 整体 F 检验 | 3357.73 | 5.2812e-69 |

## 4. 结论
- 手动计算结果与 sklearn、statsmodels 完全一致，验证了公式的正确性。
- 估计量的偏差非常小（0.0005），方差约为 0.0012，表明最小二乘估计具有无偏性和有效性。
- 假设检验的 p-value 远小于 0.05，拒绝 $$\beta_1 = 0$$ 的原假设，说明 $X$ 对 $y$ 有显著影响。
- 整体 F 检验同样显著，模型整体有效。
- 分布图显示 $$\hat{\beta}_1$$ 近似正态分布，符合大样本理论。
