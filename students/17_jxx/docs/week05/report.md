# Week 05 实验报告：看见不可见的——协方差与多重共线性

## 一、实验背景
在多元线性回归理论中，系数估计量的协方差矩阵为：
$$
\text{Var}(\hat{\beta}) = \sigma^2 (X^T X)^{-1}
$$
当特征之间存在严重多重共线性时，矩阵 \(X^T X\) 接近奇异，其逆矩阵元素会剧烈放大，直接导致系数估计方差爆炸。
本次实验通过蒙特卡洛模拟，从数值和可视化角度验证这一理论，观察多重共线性对协方差矩阵和系数估计分布的影响。

## 二、实验设计
- 真实参数：\(\beta = [5.0, 3.0]^T\)
- 噪声标准差：\(\sigma = 2.0\)
- 样本量：\(n = 1000\)
- 模拟次数：1000 次
- 实验 A：\(\rho=0.0\)，特征相互独立
- 实验 B：\(\rho=0.99\)，特征高度共线性
- 采用固定设计（Fixed Design）：特征矩阵 \(X\) 只生成一次，每次模拟仅重新生成噪声 \(\epsilon\)

## 四、思考题回答
当 \(X_1\) 和 \(X_2\) 高度正相关（\(\rho=0.99\)）时，两个变量信息高度重叠，可以互相替代。
我们可以把模型对 \(y\) 的解释能力看作一个**总体预算**：

- 如果 \(\hat{\beta}_1\) 被估大，它占用了更多预算，为了保持预测值不变，\(\hat{\beta}_2\) 必须变小；
- 如果 \(\hat{\beta}_1\) 被估小，预算出现剩余，\(\hat{\beta}_2\) 必须变大来补偿。

这种“此消彼长”的预算分配机制，使得 \(\hat{\beta}_1\) 和 \(\hat{\beta}_2\) 呈现强烈负相关，在散点图上表现为一条倾斜的细长椭圆。

## 五、实验结论
1. 多重共线性会显著放大回归系数的方差，使系数估计极不稳定。
2. 高度共线性的特征会导致系数估计呈现强烈负相关，表现为椭圆型分布。
3. 经验协方差矩阵与理论协方差矩阵高度吻合，验证了线性回归系数方差公式的正确性。
4. 固定设计下的蒙特卡洛模拟能有效、稳定地观察系数估计的统计性质。
