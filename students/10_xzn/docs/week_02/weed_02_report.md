# 线性回归实验报告 (Week 02)

## 1. 实验设置
- **样本量 (n)**: 100
- **模拟次数 (n_sim)**: 100
- **真实参数**: $\beta_0 = 1, \beta_1 = 2$
- **误差项**: $\epsilon \sim N(0, 1)$

## 2. 估计方法对比
通过 100 次蒙特卡洛模拟，得到 $\beta_1$ 的平均估计偏差（Bias）：

| 方法 | 平均估计值 | 偏差 (Bias) |
| :--- | :--- | :--- |
| **Manual (公式)** | 1.9982 | -0.0018 |
| **Statsmodels** | 1.9982 | -0.0018 |
| **Sklearn** | 1.9982 | -0.0018 |

> **结论**: 三种方法计算结果完全一致，平均偏差趋于 0，证明了 OLS 估计量的无偏性。

## 3. 统计检验与方差分析
- **平均 P-Value**: 远小于 0.05（通常为 $10^{-80}$ 量级），说明模型在统计上极其显著。
- **$\beta_1$ 估计方差**: 理论方差与模拟估算的 `manual_var` 高度吻合。
